# SAP SE Investment Research Overview

This document provides a structured, investment-grade due diligence report on SAP SE.

## Executive Summary

| Attribute | Value |
| :-- | :-- |
| **Company** | SAP SE (NYSE: SAP / ETR: SAP) [^1][^2] |
| **Current Price** | ~\$170.76 (May 1, 2026) [^3] |
| **Market Cap** | ~\$169.77B–\$201.31B (May 2026) [^4][^5] |
| **Sector** | Enterprise Software / Cloud Applications [^1] |
| **2025 Revenue** | €36.8B (+8% / +11% at constant currency) [^6][^7] |
| **Cloud Revenue** | €21.0B (+23% / +26% constant currency; now largest revenue stream) [^6][^7] |
| **Operating Profit** | €9.6B IFRS (+111% YoY) [^6][^7] |
| **Free Cash Flow** | €8.2B (nearly doubled from €4.2B) [^6][^7] |
| **Share Buyback** | €10B two-year repurchase program announced [^6][^8] |

**Investment Thesis Snapshot**: SAP is experiencing an "unstoppable modernization cycle" as enterprises migrate from legacy ECC to S/4HANA Cloud before the 2027 end-of-support deadline, with AI (Joule copilot) now overtaking migration urgency as the primary customer driver. Cloud ERP Suite revenue grew 28% (+32% constant currency), demonstrating successful cloud transition.[^6][^9][^10]

## Business Model Deep Dive

### Core Revenue Streams

| Revenue Source | 2025 Amount | Growth |
| :-- | :-- | :-- |
| Cloud Revenue | €21.0B | +23% (+26% cc) [^6][^7] |
| Total Revenue | €36.8B | +8% (+11% cc) [^6][^7] |
| Cloud ERP Suite | N/A | +28% (+32% cc) [^6] |

*   **Cloud backlog**: €77B total (+30% constant currency); Current cloud backlog +25% constant currency[^6]
*   **Segments**: Applications, Technology & Services (primary); Qualtrics; Services[^11][^12][^13]

### Customer Base & Stickiness

*   **Switching costs**: Extremely high—replacing SAP requires multi-year, high-risk core business re-engineering[^14][^15]
*   **Retention rate**: >99% for core ERP products[^14]
*   **55% of organizations** have initiated S/4HANA migration; only 34% fully completed[^9]

### Geographic Exposure

*   **Presence**: 180 countries; 109,973 employees[^16][^1][^17]
*   **Regions**: EMEA, Americas, APJ (Asia Pacific including Japan)[^17]

## Economic Moat Analysis

| Moat Characteristic | Assessment |
| :-- | :-- |
| **Moat Type** | **Strong Moat** [^14][^15][^18] |
| **Primary Source** | **Switching Costs** (core of moat) [^14] |
| **Switching Cost Evidence** | Multi-year replacement; hundreds of millions in cost; significant operational risk [^14] |
| **Brand Advantage** | 50+ year reputation; synonymous with ERP (\#1 market share) [^14][^19] |
| **Scale Advantage** | World's largest ERP vendor; massive partner/consultant ecosystem [^1][^19] |
| **Moat Sustainability** | Low disruption risk; migration cycle reinforces moat through 2027+ [^15][^10] |

Morningstar classifies SAP as having a **broad competitive advantage** based on switching costs, particularly for ERP and HANA database (~75% of revenue).[^18]

## Technological Moat Assessment

### AI Strategy & Innovation

| Factor | Details |
| :-- | :-- |
| **AI Copilot** | **Joule** (embedded generative AI); in 2/3 of Q4 2025 cloud orders [^6][^20] |
| **AI Integration** | Deeply embedded across Finance, Supply Chain, HR, Analytics by 2026 [^20] |
| **AI as Growth Driver** | SAP Business AI is "main driver for growth" per CEO [^6] |
| **High-Impact AI Use Cases** | Finance close/forecasting, supply chain exception management, autonomous procurement [^20] |
| **Platform** | SAP Business Data Cloud unifies data sources for AI [^20] |

### R&D & Technology Leadership

*   S/4HANA is AI foundation for autonomous operations, not just technical upgrade[^20]
*   Embedded AI technologies enhance user productivity with practical use cases[^21]

## Competitive Landscape

| Competitor | Market Position | Key Differentiator vs SAP |
| :-- | :-- | :-- |
| **Oracle** | Oldest rival; \#1 database market share [^19] | Broader tech stack (OCI infrastructure + Fusion Cloud ERP); larger revenue (~\$53B vs ~€33B) [^19] |
| **Microsoft** | Diversified techsuperpower [^19] | Azure + Dynamics 365 + Office 365; immense scale & integrated platform from IaaS to SaaS [^19] |
| **Salesforce** | CRM leader | SAP growing 3X faster than Salesforce in cloud apps [^22] |
| **Workday** | HCM/Finance cloud | SAP growing 7X faster than Workday [^22] |

**SAP's Competitive Position**: Undisputed ERP market leader; growing much faster than Top 10 cloud competitors (3X Oracle, 3X Salesforce, 7X Workday)[^22]

## Risk Factor Matrix

| Risk Category | Specific Risks |
| :-- | :-- |
| **Execution Risk** | Cloud transition margin dilution; system integrator bottleneck for S/4HANA deployments [^9] |
| **Competition** | Oracle winning cloud ERP deals; Microsoft's integrated platform advantage [^19] |
| **Legacy Transition** | ECC mainstream support ends Dec 2027; only costly extended maintenance until 2030 [^24] |
| **Migration Complexity** | Only 34% of organizations fully completed S/4HANA transition despite 55% initiating [^9] |
| **Macro Sensitivity** | Enterprise IT spending cyclicality; large deals may delay in recession [^19] |
| **Regulatory/ESG** | CSRD compliance complexity (though SAP provides solutions) [^25][^26] |
