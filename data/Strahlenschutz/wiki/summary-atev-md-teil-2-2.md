# Atomrechtliche Entsorgungsverordnung - AtEV

This document summarizes technical annexes and detailed requirements derived from the *Verordnung über Anforderungen und Verfahren zur Entsorgung radioaktiver Abfälle* (AtEV). It provides highly specific guidelines for the characterization, handling, and disposal of various types of radioactive waste, including spent fuel and vitrified materials.

## Waste Characterization Parameters
The document specifies parameters required for characterizing waste, which may include:
*   **Unit/Indicator:** Unit, or 'x' indicates if required for the container, or for irradiated fuel elements, or in the case of pellet fuel elements for fuel element cans [AtEV.md [Teil 2/2].md].
*   **Mass of Uranium (U-233, U-235):** Mass before irradiation, covering X [AtEV.md [Teil 2/2].md].
*   **Average Burnup:** X [AtEV.md [Teil 2/2].md].
*   **Date of Discharge:** X [AtEV.md [Teil 2/2].md].

## Detailed Waste Stream Descriptions
The annexes provide specific data points for different waste streams:

### 1. Spent Fuel Elements (Brenn-elemente)
For spent fuel elements, the following data points are required:
*   Standzeit24 X [AtEV.md [Teil 2/2].md].
*   Entladedatum X [AtEV.md [Teil 2/2].md].
*   Mittlerer Entladeabbrand X [AtEV.md [Teil 2/2].md].
*   Anzahl der unterzogenen Reaktorzyklen X [AtEV.md [Teil 2/2].md].
*   Abbrandzuwachs je Zyklus X [AtEV.md [Teil 2/2].md].
*   Mittlere Brennelementleistung X [AtEV.md [Teil 2/2].md].
*   Peaking-Faktor X [AtEV.md [Teil 2/2].md].
*   Oxidschichtdicke am Brennstab, Auslegungswert, axial und umfangs-gemittelt25 X [AtEV.md [Teil 2/2].md].
*   Isotopenzusammensetzung des U und Pu zum Zeitpunkt der Entladung aus dem Reaktor X [AtEV.md [Teil 2/2].md].

### 2. Pellet Fuel Elements (Kugelbrennelemente)
For pellet fuel elements, the required data includes:
*   Anzahl der Brennelementkannen pro Behälter X [AtEV.md [Teil 2/2].md].
*   Identifikationsnummer der Brennelementkanne X [AtEV.md [Teil 2/2].md].
*   Leermasse der Brennelementkanne X [AtEV.md [Teil 2/2].md].
*   Position der Brennelementkanne im Behälter X [AtEV.md [Teil 2/2].md].
*   Anzahl Kugeln pro Brennelementkanne X [AtEV.md [Teil 2/2].md].
*   Material der Brennelementkanne X [AtEV.md [Teil 2/2].md].
*   Gesamtmasse der Brennelementkanne X [AtEV.md [Teil 2/2].md].
*   Mittlerer Abbrand der Kugelbrennelemente in einer Brennelementkanne X [AtEV.md [Teil 2/2].md].
*   Höchster Abbrand einer Brennelementkugel in einer Kanne X [AtEV.md [Teil 2/2].md].
*   Abbrand-Verteilung der Kugeln in einer Brennelementkanne X [AtEV.md [Teil 2/2].md].
*   Anzahl der Kugeln nach Typ (HEU/LEU/ Moderator/Absorber) X [AtEV.md [Teil 2/2].md].
*   Maximales Kernbrennstoffinventar eines Kugelbrennelements (U-233, U-235, Pu-239, Pu-241, Th-232) X [AtEV.md [Teil 2/2].md].

### 3. Vitrified Radioactive Waste (Verglaste radioaktive Abfälle)
For vitrified radioactive waste, the following parameters are detailed:
*   **Fixing Agent Quality:** Quality of the fixing agent (fixation in glass with optimized chemical composition) [AtEV.md [Teil 2/2].md].
*   **Glass Matrix Composition:** X [AtEV.md [Teil 2/2].md].
*   **Mixing Ratio:** X – Mengenverhältnis Abfall : Fritte : Zuschlagstoffe [AtEV.md [Teil 2/2].md].
*   **Waste Oxide Content:** X – Anteil der Abfalloxide (Ist-Beladung) [AtEV.md [Teil 2/2].md].
*   **Glass Frite Chemical Composition:** X – Chemische Zusammensetzung der Glasfritte [AtEV.md [Teil 2/2].md].
*   **Mixing:** – Durchmischung [AtEV.md [Teil 2/2].md].
*   **Incorporation:** – Verglasung/Einbindung des Abfalls ja/nein [AtEV.md [Teil 2/2].md].
*   **Homogeneity:** – Homogene Aktivitätsverteilung ja/nein [AtEV.md [Teil 2/2].md].
*   **Volume/Mass:** – Volumen Kokille X, – Masse Glaskörper X [AtEV.md [Teil 2/2].md].
*   **Glass Product State:** Details include Density [g/cm3], Transformation Temperature [°C], and storage below transformation temperature (yes/no) [AtEV.md [Teil 2/2].md].

### 4. Compacted Radioactive Waste (Kompaktierte radioaktive Abfälle)
For compacted waste, the required measurements are:
*   Position of the ingot in the container X [AtEV.md [Teil 2/2].md].
*   Ingot material X [AtEV.md [Teil 2/2].md].
*   Specified pressing pressure X [AtEV.md [Teil 2/2].md].
*   Weight of the pressling X [AtEV.md [Teil 2/2].md].
*   Diameter of the pressling X [AtEV.md [Teil 2/2].md].
*   Height of the pressling X [AtEV.md [Teil 2/2].md].
*   Were casings and structural parts compacted together with technology waste (yes/no) [AtEV.md [Teil 2/2].md].
*   Cartridge material X, Weight of the empty cartridge X, Number of presslings X [AtEV.md [Teil 2/2].md].
